# NDX-BALL-NEO
A modern arcade brick-breaker game built with performance and extensibility in mind.

## Features

- **Dynamic Gameplay**: High-speed physics and smooth controls.
- **Power-ups**: Variety of enhancements including multi-ball, fireballs, and paddle expansions.
- **Level Editor**: Built-in tools to create and share custom stages.
- **Cross-Platform**: Optimized for both desktop and mobile browsers.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/username/ndx-ball-neo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ndx-ball-neo
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

To start the development server:

```bash
npm run dev
```

To build for production:

```bash
npm run build
```
