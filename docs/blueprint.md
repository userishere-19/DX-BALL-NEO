# **App Name**: Neon Breakout

## Core Features:

- Paddle Control: Enable intuitive paddle movement for ball bouncing.
- Brick Destruction: Implement dynamic brick destruction mechanics.
- Score Tracking: Track player's score in real-time.
- Level Progression: Enable sequential progression through multiple levels.

## Style Guidelines:

- Background color: Dark charcoal gray (#1A1A1A) to emulate a retro arcade environment.
- Primary color: Vibrant magenta (#FF00FF) for the paddle and ball to create a striking neon effect.
- Accent color: Electric blue (#7DF9FF) for brick outlines and score display, complementing the primary color with a cool, high-contrast hue.
- Font: 'Space Grotesk' sans-serif for a techy arcade feel for all UI elements.
- Use simple, glowing icons for power-ups and menu options.
- Implement retro-style animations for brick destruction and power-up effects.
- Classic arcade layout with score and level indicators at the top.
